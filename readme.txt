SpotLocate allows super-easy inclusion of a location-based map with your post. No hassle, no frills, just maps.

== Description ==

This plugin was developed for use on [http://greatdivide.nl](greatdivide.nl), to easily add an address or GPS coordinates to a post and provide a unified way of styling the resulting image when injected into a post.

== Installation ==

1. Upload the `spotlocate`-folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
